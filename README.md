# c-cli-gpp-make-muli-file-hello-world

## Description
This is a POC project to demonstrate a
make as a build tool that links multiple header and source files.

## Tech stack
- g++
- make
- c

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
